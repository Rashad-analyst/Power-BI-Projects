This Power BI project analyzes survey responses from data professionals across multiple countries.
The dashboard provides insights into job roles, salaries, tools, career challenges, and overall satisfaction in the data industry.

Key Insights & Metrics Included

✔ Distribution of survey takers by country
✔ Average salary by job title (Data Analyst, Data Engineer, Data Scientist, etc.)
✔ Most popular programming languages
✔ Difficulty breaking into the data field
✔ Job satisfaction & salary happiness scores

Tools & Skills Used

Power BI — Data modeling, relationships, DAX measures

Data Cleaning — Power Query

Visualization — KPI Cards, Bar Charts, Slicers, Filters

Exploratory Data Analysis (EDA)

Business Value

This dashboard helps:

Understand hiring & salary trends in the data industry

Compare career paths across different job roles

Identify skills and tools most commonly used

Support decision-making for career planning & workforce analytics
